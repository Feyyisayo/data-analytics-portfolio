## SQL — Employee & Department Analysis

**Tools:** SQL Server (T-SQL)  
**Dataset:** Microsoft AdventureWorks (HumanResources schema)

### What it covers
- Multi-table JOINs across employee, department, and shift data
- Filtering active records using NULL checks (e.g. current department assignments)
- Aggregations — COUNT, MIN, GROUP BY — to summarise workforce data
- Identifying employees with multiple department histories
- Finding departments with zero active headcount using LEFT JOIN + NULL filter

### Business questions 
1. List employees with their job titles
2. Show each employee's current department
3. Count employees per department
4. Identify employees who have moved across departments
5. Show each employee's current shift
6. Find employees with no department history
7. Find the earliest hire date per department
8. List employees hired in a specific year with their department
9. Count department changes per employee
10. Identify departments with no current employees
